## Installation

If `RcppFaddeeva` is not available for the current R version use:

``` r
library(remotes)
remotes::install_version("RcppFaddeeva", "0.2.2")
```

You can install the development version with:

``` r
remotes::install_github("jan-meissner/spectralem")
```
